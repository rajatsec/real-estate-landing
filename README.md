# 🏠 Premium Real Estate Landing Page | EstateFlow

A high-end, responsive landing page for real estate agencies and property developers to showcase luxury listings and capture high-intent leads.

---

## 🚀 Overview
**EstateFlow** is built with a focus on trust, visual impact, and intuitive property exploration. This project utilizes a grid-based architecture to display properties, ensuring that the visual content remains the hero of the experience.

---

## ✨ Features
- 🏙️ **Dynamic Property Grids:** Interactive displays of available listings.
- 🖼️ **Immersive Hero Section:** High-impact visual storytelling.
- 📱 **Adaptive UI:** Seamless experience on Desktop, Tablet, and Mobile.
- 🗺️ **Integrated Maps:** (Planned) Mapbox/Google Maps integration for property locations.
- 🔒 **Secure Forms:** Clean, validated contact forms for inquiries.

---

## 🛠️ Tech Stack
- **Framework:** [React](https://reactjs.org/) (for interactive components)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) (Smooth scrolls and reveals)
- **Icons:** [Lucide-React](https://lucide.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)

## 🚀 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rajatsec/real-estate-landing.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start development server:**
   ```bash
   npm run dev
   ```
